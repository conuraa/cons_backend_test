# Changelog

## [1.5.0] - 2026-01-29

### Added
- Защита терминальных статусов (closed, resolved, cancelled) от перезаписи
- Поле display_name для менеджеров (короткое имя из 1С)
- Дисклеймер про очередь при переносе консультации
- Уведомление о завершении диалога в Telegram боте
- Отдельный DASHBOARD_API_TOKEN для безопасности

### Fixed
- Исправлен парсинг webhook при закрытии консультации
- Статус консультации обновляется в БД при закрытии через Chatwoot
- Очистка контекста пользователя при попытке писать в закрытый диалог

### Changed
- Улучшены сообщения о завершении диалога
- Убраны иконки и линии из сообщений бота
- Убрана кнопка оценки консультации
